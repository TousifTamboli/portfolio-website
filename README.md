# TOUSIF // ARCHITECTURAL_PORTFOLIO_V2

A premium, high-performance portfolio engineered for the modern web. Built with a focus on system-level aesthetics, real-time data integration, and high-fidelity animations.

## 🛠️ THE_TECH_STACK

- **Core**: React 18 + Vite (Ultra-fast HMR)
- **Styling**: Tailwind CSS (Atomic Utility First)
- **Animation**: Framer Motion (High-performance transition engine)
- **Data**: 
  - **LeetCode API**: Real-time algorithmic stats & submission tracking.
  - **Google Forms**: Serverless contact management with zero-redirect submission.
- **Icons**: Lucide React + Custom SVG Path Animations

## 🚀 CORE_FEATURES

- **Architectural Showcase**: Bento-grid project display with interactive 3D physics and conditional technical breakdowns for featured works.
- **Real-time Stats**: Live hydration of LeetCode difficulty tracking, global ranking, and contribution heatmaps.
- **Interactive Terminal**: CLI-inspired section for system diagnostics and algorithmic problem history.
- **Neural UI**: Particle-based background atmospheres and responsive SVG circuit animations.
- **Micro-Interactions**: Custom cursor-reactive logic and spring-based rotation on all project nodes.

## 📦 LOCAL_DEPLOYMENT

```bash
# Clone the repository
git clone https://github.com/TousifTamboli/portfolio-website.git

# Navigate to the system directory
cd portfolio-website

# Install dependencies
npm install

# Initialize development server
npm run dev
```
